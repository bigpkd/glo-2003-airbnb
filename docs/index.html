post /beds

Add a new bed.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • ownerPublicKey: required(string - pattern: ([A-F]|[0-9]){64})

    64-chars hexadecimal string

  • zipCode: required(string - pattern: ([0-9]){5})

    5-digit valid US zip code

  • bedType: required(one of latex, memoryFoam, springs)

    Valid bed type

  • cleaningFrequency: required(one of never, weekly, monthly, annual)

    Valid cleaning frequency

  • bloodTypes: required(array of types.BloodType)

    Array of available blood types in bed

  • lodgingMode: (one of private, cohabitatioVn - default: private)

    Valid lodging mode

  • capacity: required(integer - minimum: 1)

    Capacity of bed, must be positive and must not exceed bed type capacity

  • packages: required(array of types.BedPackage)

    Array of available packages in bed, with no duplicated package name

    Items: BedPackage

    • name: required(one of bloodthirsty, allYouCanDrink, sweetTooth)

      Valid package name

    • pricePerNight: required(number)

      Price, with a maximum of 2 decimals

Example:

{
  "ownerPublicKey": "8F0436A6FB049085B7F19AB73933973BF21276276F2EC7D122AC110BB46A3A4E",
  "zipCode": "12345",
  "bedType": "latex",
  "cleaningFrequency": "weekly",
  "bloodTypes": ["O-", "O+", "AB-"],
  "capacity": 234,
  "lodgingMode": "cohabitation",
  "packages": [
    {
    "name": "bloodthirsty",
    "pricePerNight": 99.99
    }
  ]
}

HTTP status code 201

Bed successfully added

Headers

  • location: required(string)

    Path to newly created bed

    Example:

    /beds/590041fd-afc4-4b5e-bc4d-3c65396c4bc6

HTTP status code 400

Error while adding bed

get /beds

List all Beds matching query criteria

  • Request
  • Response

Query Parameters

  • package: (one of bloodthirsty, allYouCanDrink, sweetTooth)

    Valid package name

  • bedType: (one of latex, memoryFoam, springs)

    Valid bed type

  • cleaningFreq: (one of never, weekly, monthly, annual)

    Valid cleaning frequency

  • bloodTypes: (array of string)

    Array of required blood types for bed

  • minCapacity: (types.BedCapacity)

    Minimal residual capacity for given date and number of nights

  • arrivalDate: (date-only - default: now)

    Requires minCapacity, must be set in future

  • numberOfNights: (integer - default: 3 - minimum: 1 - maximum: 90)

    Requires minCapacity

  • origin: (string - pattern: ([0-9]){5})

    5-digit valid US zip code

  • maxDistance: (integer - default: 10 - minimum: 1)

    Distance from origin in kilometer, requires origin

  • lodgingMode: (one of private, cohabitatioVn)

    Valid lodging mode

Add a new booking to bed

  • Request
  • Response

URI Parameters

  • bedNumber: required(string)

    UUID representing bed

Body

Media type: application/json

Type: object

Properties
  • tenantPublicKey: required(string - pattern: ([A-F]|[0-9]){64})

    64-chars hexadecimal string

  • arrivalDate: required(date-only)
  • numberOfNights: required(integer - minimum: 1 - maximum: 90)

    Number of nights for booking, must be between 1 and 90

  • package: required(one of bloodthirsty, allYouCanDrink, sweetTooth)

    Valid package name

  • colonySize: required(integer)

    Colony size for booking, only required if bed has a cohabitation lodging mode

Example:

{
  "tenantPublicKey": "72001343BA93508E74E3BFFA68593C2016D0434CF0AA76CB3DF64F93170D60EC",
  "arrivalDate": "2020-05-21",
  "numberOfNights": 3,
  "package": "allYouCanDrink"
}
